### Installation

1. Clone the repository:

```bash
git clone https://github.com/yakyo/face-smoothing.git
cd face-smoothing
```

2. Install the required dependencies:

```bash
pip install -r requirements.txt
```

3. Run `detect.py`:

```bash
python detect.py --input 0 (webcam)  --output output.mp4
                         video.mp4
                         image.jpg
```
